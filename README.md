### 今回使用しているフレームワーク
FastAPI

### 今回使用しているライブラリ
#### wikipediaapi
Wikipedia APIにアクセスするためのPythonライブラリで、Wikipediaのページ情報を取得するために使用します。

#### Pydantic
データのバリデーションと設定管理を行うためのPythonライブラリです。FastAPIと組み合わせて、リクエストデータのバリデーションに使用されます。
